# RomanSpectre — классификация рамановских спектров мозговой ткани

Решение задачи хакатона: определение экспериментальной группы образца (контроль, эндогенная или экзогенная экспрессия HSP70) по спектру комбинационного рассеяния.

---

## Постановка задачи

- **Вход:** спектры рамановской спектроскопии биологических тканей (интенсивность от волнового числа, область ~930–2000 см⁻¹).
- **Выход:** один из трёх классов — **control**, **endo**, **exo**.
- **Дополнительно:** учёт региона мозга (cortex, striatum, cerebellum и др.) и параметра центра (1500/2900).

---

## Решение

### Предобработка спектров (по методичке)

- Коррекция фона: вычитание минимума по каждому спектру.
- Подавление шума: сглаживание Савицкого–Голея (окно 11, порядок 3).
- Нормировка: L2 (каждый спектр — вектор единичной длины).

### Методы ML

- **Ансамбль из 4 базовых моделей:**
  - **LightGBM**, **CatBoost** (градиентный бустинг на деревьях);
  - **RidgeClassifier**, **SGDClassifier** (линейные модели).
- **Стекинг:** out-of-fold предсказания базовых моделей (вероятности по классам) подаются в **мета-модель** — логистическая регрессия, дающая итоговый класс и вероятности.
- **Валидация:** Group K-Fold по `source_file`, чтобы спектры одного образца не попадали и в train, и в test (защита от утечки).

### Метрики (на тестовой выборке)

| Модель    | Accuracy | F1 (macro) | Примечание                          |
|-----------|----------|------------|-------------------------------------|
| LGBM      | ~0.53    | ~0.51      | —                                    |
| CatBoost  | ~0.54    | ~0.54      | Лучшая базовая модель                |
| Ridge     | —        | —          | Линейный базовый уровень            |
| SGD       | —        | —          | Линейный базовый уровень            |
| **Stacking** | **~0.54** | **~0.53** | Итоговое предсказание ансамбля      |

По классам: лучше всего распознаётся **exo** (recall ~0.81), слабее — **control** (путаница с exo); у **endo** высокий precision (~0.83), но низкий recall (~0.37).

## Стек и зависимости

- **Данные:** спектры в виде таблицы (волновые числа × интенсивности), сбор в единый датасет — `build_dataset.py`.
- **Обучение:** Python 3, pandas, numpy, scikit-learn, LightGBM, CatBoost, scipy (для Savitzky–Golay).
- **Веб-сервис:** Flask; загрузка одного спектра (файл Wave/Intensity), выбор региона и центра, предсказание класса и вероятностей.

---

## Структура проекта

- `sample.ipynb` — полный пайплайн: загрузка данных, предобработка, разбиение по группам, обучение ансамбля, оценка, экспорт моделей.
- `build_dataset.py` — сбор `dataset_combined.csv` из сырых файлов спектров по папкам control/endo/exo.
- `model_export/` — сохранённые модели (нативные форматы LGBM/CatBoost + joblib для Ridge, SGD и мета-модели).
- `web/` — веб-интерфейс: загрузка файла спектра, выбор региона мозга и центра, получение предсказания.

---

## Запуск

1. **Сбор датасета:**  
   `python build_dataset.py` (при необходимости с аргументом лимита файлов на класс).

2. **Обучение:**  
   Открыть `sample.ipynb`, выполнить ячейки по порядку (включая экспорт в `model_export/`).

3. **Веб-сервис:**  
   ```bash
   cd web && pip install -r requirements.txt && python app.py
   ```  
   Открыть http://127.0.0.1:5000, загрузить файл с колонками Wave и Intensity, выбрать регион и центр — получить класс (control/endo/exo) и вероятности.

---

## Итог

Решение использует **предобработку спектров** (коррекция фона, сглаживание, нормировка), **ансамбль из LightGBM, CatBoost и линейных моделей (Ridge, SGD)** и **стекинг с мета-моделью** при строгой **Group K-Fold** валидации. Достигается **accuracy ~54%** и **F1 macro ~0.53** на трёх классах; развёртывается как веб-сервис для предсказания по одному загруженному спектру.
